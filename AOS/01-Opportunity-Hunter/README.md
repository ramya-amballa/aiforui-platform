# Opportunity Hunter

AI employee #1. Finds, logs and scores every credible revenue
opportunity for AI for U&I, so nothing gets missed because no one
had time to look.

## Files

- `operating-manual.md` — mission, daily workflow, sources, scoring criteria, daily output, success metrics
- `scoring-model.md` — the weighted rubric behind every score in `opportunities.json`
- `opportunities.json` — the live opportunity log
- `companies.md` — organisations encountered while sourcing opportunities (full company intelligence lives in `06-CRM/company-intelligence.json`)
- `proposal-template.md` — the first-touch note or proposal starter sent for a qualified opportunity
- `daily-report-template.md` — the day's findings, handed off to `07-Daily-Brief`

Start with `operating-manual.md`.
