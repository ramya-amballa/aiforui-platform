# Sales Director

AI employee #4. Owns every recruiter, consulting firm, prospect and
client interaction after the first touch. Where `opportunity-hunter`
finds an opportunity and `08-Revenue-Hunter` decides it's worth
pursuing, Sales Director makes sure nothing pursued is ever allowed to
go quiet. It recommends a follow-up for every open relationship every
day and drafts the outreach itself.

## Files

- `operating-manual.md` — mission, daily workflow, and how it fits the
  rest of AOS
- `follow-up-priority-model.md` — how "which relationships need
  attention today" gets decided
- `outreach-draft-template.md` — the structure every personalised
  outreach draft follows

Sales Director reads and writes `06-CRM/company-intelligence.json`
directly. It does not keep a separate contact database.

Start with `operating-manual.md`.
