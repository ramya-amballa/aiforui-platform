# CEO Advisor

AI employee #9, and the only one with no output-facing role. CEO
Advisor never writes content, drafts outreach, scores a lead, or logs
a regulation. It reads what every other employee has already produced
and answers one question, every morning: **what is the highest-ROI
thing Ramya should do today?**

It exists because seven employees each optimising their own function
(sourcing, pipeline, content, product, sales, CRM, market watch) can
each be individually right and still leave the founder without a clear
single answer to "what should I actually do in the next hour." CEO
Advisor is the aggregation layer that turns seven streams of output
into one instruction.

## Files

- `operating-manual.md` — mission, daily workflow, and how it fits the
  rest of AOS
- `decision-model.md` — how value, effort, probability and urgency are
  weighed against each other to pick a single top action
- `daily-recommendation-template.md` — the exact output format

CEO Advisor reads from every other employee's files. It writes to
nothing except its own daily recommendation, which `07-Daily-Brief`
then surfaces as its lead item.

Start with `operating-manual.md`.
